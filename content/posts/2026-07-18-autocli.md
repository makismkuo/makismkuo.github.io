## 项目简介

[AutoCLI](https://github.com/nashsu/AutoCLI)（原名 opencli-rs）是一个纯 Rust 重写的命令行工具，让你**一个命令抓取任何网站的数据**。覆盖 Twitter/X、Reddit、YouTube、HackerNews、Bilibili、知乎、小红书等 55+ 网站，内置 333 个预置命令——无需写任何爬虫代码，无需管理 Cookie 和 Token，下载即用。当前 ⭐2878。

它的前身 [OpenCLI](https://github.com/jackwener/opencli)（TypeScript）已有 5600+ ⭐，AutoCLI 用 Rust 完整重写后，性能提升 12 倍，内存降低 10 倍，二进制仅 4.7MB，零运行时依赖。

## 核心功能

**55+ 网站即开即用。** 装好就能跑 `autocli hackernews top --limit 10` 看 Hacker News 头条，`autocli zhihu hot` 刷知乎热榜，`autocli bilibili hot --limit 20` 查 B 站热门。输出支持 table、JSON、YAML、CSV、Markdown 多种格式，方便 pipe 给 jq 或其他工具处理。

**零运行时依赖。** 没有 Node.js、没有 Python、没有 npm install——就是一个 4.7MB 的静态二进制文件，下载扔进 PATH 就能用。对比 OpenCLI 需要 Node.js 20+ 和完整的 node_modules（约 50MB），这差距肉眼可见。

**Chrome 会话复用。** 对于需要登录的站点（Twitter、知乎、小红书），AutoCLI 通过 Chrome 扩展复用它浏览器中已登录的会话，无需手动配 Token 或 Cookie。装一次扩展，所有站点无缝使用。

**AI 原生集成。** `autocli explore` 自动分析网站 API 结构，`autocli generate --ai` 用 LLM 自动生成适配器，无需手写任何代码。生成的适配器可以上传到 autocli.ai 云端共享。AutoCLI 还提供 AI Agent 技能安装命令——`npx skills add https://github.com/nashsu/autocli-skill`，让你在 Claude Code、Cursor、Copilot 等 AI 工具中直接调用它的能力。

**外部 CLI 透传。** 可以用 `autocli register gh` 把 GitHub CLI 注册进来，之后 AI Agent 就能自动发现并调用你本地的 gh、docker、kubectl 等工具。

**视频和文章下载。** `autocli` 还能下载视频（集成 yt-dlp）和文章（转 Markdown，图片本地化），适合做内容存档或离线阅读。

## 为什么值得关注

AutoCLI 解决了一个真实痛点：**数据获取的碎片化**。每个网站有自己的 API、认证方式和输出格式。开发者写个爬虫要折腾反爬、Cookie 续期、结果解析；AI Agent 想要实时数据更是处处受限。

AutoCLI 用一套统一的 CLI 接口把这一切抹平了。你不需要记住不同网站的 API endpoint 或认证机制，只用 `autocli <site> <command> [options]` 就行。对 AI Agent 来说尤其有价值——一个工具调用就能获取跨 55 个平台的实时信息。

性能数据也很硬：纯 Rust 编译的单二进制，无外部依赖，实测比 TypeScript 版本快 12 倍（Bilibili 热榜从 20s 降到 1.66s），内存从 99MB 降到 15MB。这不是"差不多快"，是量级差距。

## 简单示例

```bash
# 安装（macOS/Linux）
curl -fsSL https://raw.githubusercontent.com/nashsu/autocli/main/scripts/install.sh | sh

# 看 Hacker News 头条
autocli hackernews top --limit 5

# 看知乎热榜
autocli zhihu hot --limit 10 --format json

# 搜索 Twitter
autocli twitter search "rust lang" --limit 10

# 看 GitHub 趋势（需配置 Token）
autocli github trending --language rust

# 输出为 Markdown
autocli hackernews top --limit 10 --format markdown

# 运行诊断检查
autocli doctor
```

更强大的 AI 用法——在你的 Agent 配置里加一行 `autocli list`，AI 就能自动发现所有可用命令并调用它们。
