## 背景

需要生成一张特定人物的照片，但每次让 AI 自由发挥风格不统一。LoRA 就是解决这个问题的——用少量样本训练一个人物的专属模型。

## 关键发现

### 1. 训练照片决定一切

清晰的正脸照 + 光线均匀的照片效果最好。带杂乱的背景、模糊、侧脸过度的照片会直接拉低出图质量。

精选 10-15 张高质量照片，比随便扔 30 张进去效果好得多。

### 2. API 调用必须用模型版本

```
POST /v1/models/{owner}/{name}:{version}/predictions
```

不要用 `lora_weights` URL 参数——CDN 的临时链接几小时后就会过期，返 404。

### 3. 触发词

训练时指定触发词（比如 `TOK`），出图时在 prompt 里带这个词就能触发 LoRA 效果。

## 成本

每次出图大概 $0.05，比 Midjourney 便宜，效果接近。适合做人物 IP、角色设计、产品图。
