## 项目简介

[Fallow](https://github.com/fallow-rs/fallow) 是一个用 Rust 写的代码库智能分析工具，专为 TypeScript 和 JavaScript 项目设计。一个二进制文件就能检测出 unused code、circular dependencies、code duplication、complexity hotspots、boundary violations——而且不需要 Node.js 运行时，不依赖 TypeScript 编译器。

```bash
npx fallow          # 零配置，全量跑
npx fallow audit    # Gate 只改动的部分
```

目前在 GitHub 上有 **4121 star**，MIT 许可，由 Rust + oxc 实现。安装即用，npm install --save-dev fallow 即可。

## 核心功能

Fallow 覆盖了代码质量管理的大部分维度，全部通过一个 CLI 统一暴露：

- **死代码检测**：unused exports、unused dependencies、unused files，精确到 import 图级别
- **循环依赖**：把模块间的循环引用路径可视化，知道哪里需要拆分
- **代码重复**：检测 copy-paste（支持跨文件），输出 clone groups
- **复杂度热区**：圈复杂度分析，找出最需要重构的函数/模块
- **边界违规**：模块划分被打破的地方（比如 infrastructure 层引了 UI 层）
- **设计系统漂移**：检测样式 token 不一致

输出格式极其丰富：human（终端报告）、json（机器契约类型安全）、sarif（GitHub Code Scanning）、markdown、compact（一行一条）、codeclimate（GitLab 格式）、GitHub annotations 和 summary。每种格式都有对应的 exit code 约定——0 是干净，1 是有发现但执行成功，2 才是真正错误。

## 为什么值得关注

**1. 性能极好，单二进制部署**

因为用 Rust + oxc 做解析，Fallow 不需要启动 Node.js 或 TSC 就能分析。实测在 vitest 仓库跑 audit 只用 1 秒多。CI 集成简单到一句话——GitHub Actions 的 `fallow-rs/fallow@v3` 开箱即用。

**2. 对 AI Agent 原生友好**

这是我最欣赏的设计——Fallow 有内置的 MCP server：`npx fallow-mcp`，所有分析功能以 MCP tool 形式暴露给 Claude Code、Codex 等 AI 编码助手。还提供了 `npx fallow init --agents` 生成 AGENTS.md，让 AI 知道怎么调用。对于一个靠 AI Agent 写代码的团队，这意味着 CI 流程里自动 running fallow audit 并把结果塞回 PR 评论——完全不需要人类介入。

**3. 渐进式采用设计**

Fallow 的 audit 模式很聪明：它只对 PR 引入的改动报错，已有的老问题不阻塞 day-1 采用。`--save-baseline` 可以存档当前 repo 状态，之后跑 audit 只报新问题。这对于接入一个十万行历史项目来说，是决定能否落地的关键设计。

**4. 迁移友好**

`npx fallow migrate` 可以自动把 knip、jscpd、stylelint 配置翻译成 fallow 配置——迁移不是重来，而是平移。

## 简单示例

```bash
# 安装
npm install --save-dev fallow

# 全量分析（死代码 + 重复 + 健康评分）
npx fallow

# 只在 PR 变动的文件上跑
npx fallow audit

# JSON 输出给 CI 脚本
npx fallow audit --format json --quiet

# CI 集成（GitHub Actions）
# .github/workflows/ci.yml
# - uses: fallow-rs/fallow@v3

# 给 AI Agent 配 MCP
# .claude/settings.json 加：
# { "mcpServers": { "fallow": { "command": "npx", "args": ["fallow-mcp"] } } }

# 迁移现有配置
npx fallow migrate  # 从 knip/jscpd/stylelint 自动转
```

## 总结

Fallow 解决了一个每个中大型 JS/TS 项目最终都会遇到的问题——代码库失控不知道哪里能删。它用 Rust 把分析速度推到极致，用 MCP 和 agent skill 让 AI 也能参与，用 audit 模式让渐进式落地成为可能。如果你维护一个 TypeScript monorepo、或者团队依赖 AI Agent 写代码，Fallow 是当前最值得加入工具链的项目之一。
