## 关于这个博客

Fanssen Notes 是凡森科技（Fanssen Technology Co., Ltd.）的技术博客。我是 SEB，公司的负责人。

这个博客的存在理由很简单：每天都会花一些时间研究 GitHub 上的开源项目，从技术框架到 AI 工具，发现了什么有意思的东西就部署试试看。把这段时间积累的东西写下来，对我来说是一个整理思路的过程，对你来说可能在某些场景下也能用上。

## 这里会有什么

目前规划了三个方向：

第一个是**项目实战**。从选型理由到部署过程，从踩坑记录到最终效果，把完整的过程写下来。每篇文章对应一个具体的项目，这样查起来也方便。

第二个是**技术笔记**。学习新知识点的时候，用文章的形式把理解写下来。写出来和想明白之间还有一段距离，这算是个人学习的方法。

第三个是**日常随笔**。一些跟技术没什么关系的内容——想法、观察、或者没什么特定主题的文字。

## 内容来源

这个博客有一个比较特别的机制：一部分文章是人工写的（比如这篇），另一部分是 AI Agent 在部署项目时自动生成的。

具体操作是这样的：我或者 AI Agent 在研究一个项目之后，会把过程中的关键发现整理成文章，经过审核后发布。每篇文章背后都有真实的部署过程，不是凭空编造的内容。

## 联系方式

博客的内容会同步在 blog.aibestapp.top 和 GitHub Pages 上发布。

如果你对某个项目有自己的经验，或者发现了文章中的问题，可以到我的 GitHub 仓库提交 issue。交流本身是有价值的。

## 关于 Fanssen Tech

凡森科技成立于 2025 年，主要做 AI 产品开发和技术咨询。目前的产品方向包括 AI 应用开发、自动化工具、技术内容输出等。公司还在早期阶段，很多事情都在摸索中。

这个博客既是为产品服务的——让读者了解我们的技术能力，也是个人持续学习的一个记录。希望能长久更新下去。
