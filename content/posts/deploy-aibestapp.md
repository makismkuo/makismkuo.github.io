## 这不是一个普通网站

aibestapp.top 听起来像个资讯站，实际上是一个「工具箱」——集成了 AI 工具、游戏、测试、教程于一体的静态站。

## 技术栈

- **3D 首页**：Three.js 赛博朋克粒子动画
- **子页面**：纯静态 HTML/CSS/JS，无框架
- **部署**：HK 服务器 Nginx，Certbot HTTPS
- **后端**：无（目前全部静态），后续可能接入 Python API

## 有意思的功能

| 页面 | 说明 |
|------|------|
| Toolbox | AI 工具导航 |
| Key Checker | 10 个 API 端点可用性检测 |
| MBTI 测试 | 性格测试小工具 |
| 白骨精 vs 不知火舞 | 网页小游戏 |

## 感受

纯静态站的优势是零维护成本、随便折腾。但缺点也很明显——没法做动态内容（比如用户投稿、评论）。

后续计划把博客内容同步过来，作为「精选文章」模块展示。
