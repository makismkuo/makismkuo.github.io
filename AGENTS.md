# Fanssen Notes — Blog

## Stack
- Hugo + PaperMod theme
- Hosted on GitHub Pages: `makismkuo.github.io`
- Chinese (zh-cn)

## Commands

```bash
hugo new content posts/<slug>.md   # new post
hugo server -D                      # dev preview with drafts
hugo                                # build to public/
```

## Conventions

- Publish via `git push` to `main` — GitHub Actions auto-deploys
- Signature: **Seb** (S大写eb小写)
- Blog name: **Fanssen Notes**
- Drafts not published until `draft: false` in frontmatter
